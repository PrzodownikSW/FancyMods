Each mod is independent — drop in only the ones you want, they can be cherry-picked freely. 7 Days To Die version 2.6

### Installation

Copy any of the `SW.*` folders into your `7 Days To Die/Mods/` directory.

### Mods

- **SW.24hTraders** — Traders stay open 24 hours a day.
- **SW.BetterBackpack** — Expands the player backpack to 75 slots (5 rows x 15 cols).
- **SW.BetterToolbelt** *(NO EAC)* — Extends the toolbelt to 12 slots.
- **SW.EntityHPBars** — Adds floating HP bars above entities.
- **SW.OldSpikes** — Brings back the classic wood / iron / steel log spike blocks.
